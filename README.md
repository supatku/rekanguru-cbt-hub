# Rekanguru CBT Hub

Rekanguru CBT Hub adalah platform Ujian Berbasis Komputer (CBT) yang dirancang untuk memudahkan pelaksanaan ujian di sekolah.

## Fitur Utama
- Manajemen Ujian
- Dashboard Guru
- Hasil Ujian Instan
- Antarmuka Ramah Pengguna

## Pengembangan Lokal

Ikuti langkah-langkah berikut untuk menjalankan proyek secara lokal:

```sh
# 1. Instal dependensi
npm install

# 2. Jalankan server pengembangan
npm run dev
```

## Teknologi
- Vite
- React
- Tailwind CSS
- Shadcn/UI
